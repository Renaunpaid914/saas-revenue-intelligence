## 🧭 How the Analysis Works

The project rebuilds customer history from snapshot data. That means it looks at records taken at different points in time and turns them into a timeline.

It then uses SQL and Python to:

- Group customers by month
- Measure recurring revenue
- Track when customers leave
- Compare cohorts
- Find average revenue per user
- Estimate lifetime value
- Show how much revenue comes from the top customers

This makes it easier to see the shape of the business, not just the final numbers.
